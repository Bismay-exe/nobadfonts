[![license](https://img.shields.io/npm/l/nobadfonts-cli.svg)](LICENSE)
[![downloads](https://img.shields.io/npm/dm/nobadfonts-cli.svg)](https://www.npmjs.com/package/nobadfonts-cli)

Install high-quality, curated fonts directly from the terminal.

**NoBadFonts CLI** lets designers and developers install fonts with a single command—no manual downloads, no searching through low-quality type libraries.

---

## ✨ Features

* Curated fonts only
* One-command installation
* Fast downloads
* Clean and simple CLI
* Works on macOS, Windows, and Linux

---

## 🚀 Quick Start

Run instantly without installing:

```bash
npx nobadfonts-cli
```

Or install globally:

```bash
npm install -g nobadfonts-cli
```

Then run:

```bash
nobadfonts
```

---

## 📦 Commands

### Install a font

```bash
npx nobadfonts-cli install <font-name>
```

Example:

```bash
npx nobadfonts-cli install inter
```

---

### List available fonts

```bash
npx nobadfonts-cli list
```

---

### Search fonts

```bash
npx nobadfonts-cli search <keyword>
```

Example:

```bash
npx nobadfonts-cli search display
```
